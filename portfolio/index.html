Projetos

Site sobre Queimadas no Brasil

Um projeto desenvolvido durante meu curso de Engenharia de Software que teve como objetivo a criação de um site para conscientizar a população sobre as queimadas que ocorrem frequentemente em diversas regiões do Brasil. Esse site fornece informações detalhadas sobre as causas e consequências das queimadas, além de dados estatísticos, mapas interativos que mostram as áreas mais afetadas, e dicas de como a população pode ajudar a prevenir esses incêndios florestais. O projeto também incluiu entrevistas com especialistas e vídeos educativos para aumentar a conscientização e promover ações sustentáveis.

Site para uma Cafeteria

Como parte de um exercício prático durante meu curso de Engenharia de Software, desenvolvi um site para uma cafeteria fictícia. O objetivo desse projeto era aplicar conceitos de design de interfaces, experiência do usuário e desenvolvimento web. O site inclui páginas que descrevem o menu da cafeteria, informações sobre a localização, horário de funcionamento, e opções de contato. Além disso, foram implementadas funcionalidades interativas como a possibilidade de fazer reservas online e um sistema de feedback dos clientes. Este projeto me permitiu aprofundar meus conhecimentos em HTML, CSS e JavaScript, além de experimentar com ferramentas de desenvolvimento colaborativo e controle de versão.
